# IPP — ALL‑IN‑ONE (déployer en 1 fois)
Contenu :
- index.html (Accueil + Quiz 60×6 + Scoring ADN/CAP/Caps vécus + Visuels + Export PDF par impression)
- assets/reports.json (36 narratifs complets, prêts)
- libs/ (librairie graphique locale)
- admin.html (outil local si vous voulez regénérer reports.json plus tard)

Déploiement (GitHub Pages) :
1) Créez un dépôt vide (branche main).
2) Uploadez **tout le contenu** de ce ZIP à la racine.
3) Settings → Pages → Source = main / (root).
4) Ouvrez l’URL publique. C’est prêt.
